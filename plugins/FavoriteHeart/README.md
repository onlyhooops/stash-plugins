# FavoriteHeart

Stash 红心收藏功能插件，在卡片右上角添加红心按钮，点击后自动添加/移除指定标签（默认「精选集」）。

## 功能

- 支持图片、图库、短片卡片的红心收藏
- 兼容网格视图、预览墙视图、瀑布流增强视图（EnhancedWallView）
- 可配置收藏标签名称、启用页面、红心颜色

## 配置

通过 Stash 插件设置可配置：
- `favoriteTagName`: 收藏标签名称（默认「精选集」）
- `enableOnImages` / `enableOnScenes` / `enableOnGalleries` 等开关
- `heartColor`: 红心颜色
