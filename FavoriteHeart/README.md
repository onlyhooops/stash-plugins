# FavoriteHeart

Stash 红心收藏与不喜欢功能插件：卡片**右上角**红心 ♥️ 表示喜欢（加入「精选集」），**左上角**心碎 💔 表示不喜欢（加入「Dislike」）。喜欢/不喜欢均为可逆操作，仅通过标签实现，不影响原文件与 Stash 数据库。

## 功能

- **喜欢（红心 ♥️）**：点击右上角红心，为条目添加「精选集」标签；再次点击可取消。
- **不喜欢（心碎 💔）**：点击左上角心碎，为条目添加「Dislike」标签；再次点击可取消。（在瀑布流增强视图中为避免与左上角 1080P/4K 等规格标签重叠，心碎按钮显示在**左下角**。）
- 支持图片、图库、短片卡片；兼容网格视图、预览墙视图、瀑布流增强视图（EnhancedWallView）。
- 可配置收藏/不喜欢标签名称、启用页面、红心颜色。

## 如何「隐藏」不喜欢的条目

插件只负责打标签，不修改列表数据。要隐藏不喜欢的条目，请使用 Stash 自带的筛选：

- **在列表中不显示不喜欢的**：在场景/图片/图库列表页的筛选器中，选择「排除标签」并勾选「Dislike」。
- **只查看不喜欢的**：在侧边栏或标签页进入「Dislike」标签，即可只看到已标记为不喜欢的条目。

## 配置

通过 Stash 插件设置可配置：

- `favoriteTagName`: 收藏标签名称（默认「精选集」）
- `dislikeTagName`: 不喜欢标签名称（默认「Dislike」）
- `enableOnImages` / `enableOnScenes` / `enableOnGalleries` 等开关
- `heartColor`: 红心颜色
